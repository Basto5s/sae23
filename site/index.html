Objectifs

  • Offrir aux gestionnaires des bâtiments de l’IUT une interface conviviale et simple pour la visualisation des données capteurs
  • Exploiter les données provenant de capteurs répartis dans les bâtiments de l’IUT en proposant une interface dédiée.
  • Deux bâtiments avec chacun un gestionnaire, deux capteurs par bâtiments, Dashborad Grafana avec les 4 capteurs, conception d'une base de données Mysql, Site web dynamique hébergé sur un serveur lampp, présentation des données et métrique sous forme de tableau
